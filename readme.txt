Limit which posts your users can edit or view in the admin screen.

== Description ==

Based on the Restrict Categories plugin, Restrict User Taxonomies will allow you to easily restrict post editing, viewing and adding based on selected taxonomy terms.

**Features**

* Hide posts for users in specific taxonomies
* Prevent users to add posts in unwanted categories
* Works on ALL custom taxonomies
* Compatible with older versions of WordPress
* Lightweight - Plugin only loads functionalities in admin area
* Easy to use - Simple search interface for terms
* WPML Compatible - You only need to restrict taxonomies on your base language

== Installation ==

1. Upload restrict-user-taxonomies.zip to plugins via WordPress admin panel, or upload unzipped folder to your plugins folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to User Edit Screen to restrict your users

== Frequently Asked Questions ==

= Will this plugin work for custom taxonomies added by themes / plugins =

Yes it will. It loads late in WordPress initialization process, so all registered taxonomies will be available for restriction

= Can I restrict taxonomies based on User Roles =

For now, no. This will be implemented very soon

= How should I use this plugin with WPML =

For complete compatibility you should edit your users while viewing admin area in your default language.
